3. Respect des Licences Open-Source

Obligations à Respecter

Pour AGPLv3 (MinIO, Grafana) :

  • Si vous ne modifiez pas le code source → Aucune contrainte
  • Si vous modifiez le code source → Vous devez publier vos modifications sous AGPLv3
  • Usage commercial autorisé dans les deux cas

Pour n8n (Sustainable Use License) :

  • Vous pouvez proposer n8n dans votre offre commerciale
  • Vous ne pouvez pas créer un service concurrent direct à n8n Cloud
  • Pour OceanPhenix : Aucun problème, n8n est un composant parmi d'autres
